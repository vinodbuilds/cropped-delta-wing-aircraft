# Cropped Delta Wing Aircraft

> ✈️ Design and fabrication of a cropped delta wing RC aircraft developed as an Aero Club engineering project.

---

## 📖 Overview

This repository documents the design, fabrication, and evaluation of a **Cropped Delta Wing RC Aircraft** developed as part of an Aero Club project.

The cropped delta wing configuration was selected to improve lift generation, stability, maneuverability, and aerodynamic efficiency for radio-controlled aircraft applications.

---

## 🎯 Project Objectives

- Design a cropped delta wing aircraft
- Improve aerodynamic performance
- Enhance stability and maneuverability
- Evaluate the aircraft through practical implementation

## ✈️ Aircraft Features

- Swept-back Wing Geometry
- Improved Lift Generation
- Better High-Speed Stability
- Reduced Aerodynamic Drag
- Efficient Lift-to-Drag Ratio
